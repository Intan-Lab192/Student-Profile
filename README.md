Project Description

This project is a Student Profile Management System developed using Java Web technologies.
The system allows users to create, view, search, edit and delete student profile records stored in a database.
The application follows the MVC (Model-View-Controller) architecture and integrates HTML, JSP, Servlet, JavaBean, JDBC and Apache Derby Database.

Application Used

- HTML
- CSS
- JSP
- Java Servlet
- JavaBean
- JDBC
- Apache Derby Database
- NetBeans IDE 8.2
- GlassFish Server

Features Implemented
